后记

上面的办法只对Apache服务器有效，其他修改办法参照下文：

(一)IIS Noinput file specified

方法一： 改PHP.ini中的doc_root行，打开ini文件注释掉此行，然后重启IIS
方法二： 请修改php.ini，找到 ; cgi.force_redirect = 1 去掉前面分号，把后面的1改为0即
cgi.force_redirect = 0
